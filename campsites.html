Project: Campsite Reservations Application

Description: Command Line Application for reserving campsites at national parks.

The Campsite Reservations Application shows available campsites by date at chosen parks, and displays description/info about the site.
The user can reserve a chosen camping site and block out their dates so they're unavailable for others.

Technology Used:

Java, CLI, JDBC, PostgreSQL
