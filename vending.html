Project: Vending Machine Application

Description:

Command Line Application for Vending Machine. The Vending Machine Application displays a menu, allowing the user to 1)see all items, 2)purchase, or 3)exit.
User can deposit money, then purchase multiple items before change is processed.
Initial inventory is read in from a .csv file, and a log of sales is written to a .txt file.

Technology Used:

Java, CLI, File I/O
